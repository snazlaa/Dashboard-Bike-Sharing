# Bike Sharing Dashboard 
Dashboard interaktif untuk menganalisis pola peminjaman sepeda di Washington D.C. periode 2011–2012.

## Deskripsi Project
Project ini merupakan analisis data dari Bike Sharing Dataset yang mencakup:
- Pengaruh musim dan cuaca terhadap jumlah peminjaman sepeda
- Pola peminjaman berdasarkan hari kerja vs hari libur dan jam dalam sehari
- Clustering tingkat penggunaan sepeda menggunakan metode binning

## Setup Environment
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

## Run Streamlit App
streamlit run dashboard.py
